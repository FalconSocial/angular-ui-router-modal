### $stateProvider.modalState
> All of the regular properties available to `$stateProvider.state` are available to `.modalState`.

```js
$stateProvider.modalState('someModal', {
  url:         /** String **/
  controller:  /** String|Function **/,
  templateUrl: /** String **/,
  resolve:     /** Object **/,
  sticky:      /** Boolean **/,
  views:       /** Object **/,
  /** etc etc **/
});
```
